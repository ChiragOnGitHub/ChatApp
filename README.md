# Run Locally First Time

Clone the project

```bash
  git clone https://github.com/ChiragOnGitHub/loginnest.git
```

Go to the project directory

```bash
  cd loginnest
```

Install dependencies

```bash
  npm install
  cd server
  npm install
```

Go back to the project directory

```bash
  cd loginnest
```

Start the Website

```bash
  npm run dev
```
