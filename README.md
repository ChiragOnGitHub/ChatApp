# Run Locally First Time

Clone the project

```bash
  git clone https://github.com/ChiragOnGitHub/ChatApp.git
```

Go to the project directory

```bash
  cd ChatApp
```

Install dependencies

```bash
  npm install
  cd server
  npm install
```

Go back to the project directory

```bash
  cd ..
```

Start the Website

```bash
  npm run dev
```
